# Movie Verse 🎬

Movie Verse is a movie search web application that allows users to explore and search for movies with an interactive user interface.

## Technologies Used

- HTML
- CSS
- JavaScript

## Features

- Search movies
- Display movie posters and details
- Responsive design
- Interactive user interface

## How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. Explore Movie Verse
